# IPIX-V2
Robot seguidor de línea utilizado desde el 30/08/2025 en el interzonal de Trelew de la LNR argentina con un 3° puesto interzonal. 
El mismo posee trabaja con el microcontrolador Arduino Nano, TB6612FNG, motores ProFast de 4000rpm llantas y cubiertas JSUMO SLT20 y una regleta de sensores QTR8A genérica, similar a la de Pololu. 
Posee un switch DIP para selección de estrategias o configuraciones (empleado para cambio de velocidad según interior o exterior de la pista) y una programación basada en PID básico. 
En cuanto al código, se probó un sistema con intervalos de tiempo para el control dinámico de velocidades base/máximas o parámetros de cálculo del PID, pero debido al poco tiempo de pruebas en pista no se pudo aprovechar. Además, se está desarrollando un algoritmo con Twiddle (un algoritmo de tuning de PID con una especie de aprendizaje escalonado muy básico) que aún no fue probado en pista, con el objetivo de optimizar el tiempo de pruebas en competencia y recopilar datos de tiempo de vuelta y constantes del cálculo del PID.
Por otro lado, se está avanzando con el diseño de una tercera versión (IPIX V3) migrando a una BluePill (STM32F103C8T6), RZ7899 como drivers de motores, hueco para una turbina para mayor adherencia (la limitante actual del prototipo) y 10 sensores QRE1113 directamente soldados a placa con lectura directa por cada canal del ADC disponible.
