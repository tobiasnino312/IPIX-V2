# IPIX-V2
Robot seguidor de linea utilizado desde el 30/08/2025 en el interzonal de Trelew de la LNR argentina con un 3Â° puesto interzonal. 
El mismo posee trabaja con el microcontrolador Arduino Nano, TB6612FNG, motores ProFast de 4000rpm y una regleta de sensores QTR8A generica, similar a la de Pololu. 
Posee un switch DIP para seleccion de estrategias o configuraciones (empleado para cambio de velocidad segun interior o exterior de la pista) y una programacion basada en PID basico.
